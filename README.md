# 🍇 Site Institucional - Fazenda Villa das Videiras

Este é um site elegante, moderno e responsivo para uma fazenda de plantação, colheita e venda de uvas premium. O projeto foi estruturado utilizando HTML5, CSS3 moderno e JavaScript para interatividade.

## 📂 Estrutura do Projeto

O projeto é composto por 4 arquivos principais na raiz:
1. `index.html` - Estrutura e conteúdo do site.
2. `style.css` - Estilização, cores, tipografia e responsividade.
3. `script.js` - Comportamento dinâmico (menu mobile, animações e modal).
4. `README.md` - Documentação do projeto (este arquivo).

## 🚀 Como Executar

1. Baixe ou clone os arquivos para uma pasta no seu computador.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estão na **mesma pasta**.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador de preferência.

## ✨ Funcionalidades

- **Menu Responsivo:** Adapta-se automaticamente para celulares e tablets.
- **Seções Completas:** Apresentação da Plantação, processo de Colheita e a Loja de Vendas.
- **Formulário de Contato:** Pronto para receber interações dos clientes.
- **Efeito de Rolagem Suave:** Navegação fluida entre as seções.
